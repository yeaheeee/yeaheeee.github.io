동물상별 특징 가이드

🐶 강아지상

선하고 부드러운 인상이 특징입니다. 눈매가 둥글고 처진 편이며, 주변 사람들에게 친근함과 신뢰감을 주는 매력적인 관상입니다.

  • 특징: 둥근 코, 부드러운 턱선
  • 매력 키워드: 다정함, 활발함, 귀여움

🐱 고양이상

세련되고 도도한 분위기가 매력적입니다. 눈매가 살짝 올라가 있고 뚜렷하며, 카리스마와 신비로움을 동시에 갖춘 인상입니다.

  • 특징: 날카로운 눈매, 오똑한 코
  • 매력 키워드: 시크함, 우아함, 지적임

서비스 소개

AnimalFace AI는 구글의 Teachable Machine 기술을 기반으로 구축된 얼굴 분석 서비스입니다. 수천 장의 동물상 데이터를 학습한 인공지능 모델이 사용자의 얼굴 이미지를 분석하여 가장 유사한 동물상을 백분율로 산출합니다.

개인정보 보호 정책

저희 서비스는 사용자의 이미지를 서버로 전송하지 않습니다. 모든 인공지능 분석은 사용자의 브라우저 내에서 직접 수행되며, 분석이 끝난 사진은 즉시 파기됩니다. 사용자의 데이터를 수집하거나 저장하지 않으니 안심하고 이용하세요.
